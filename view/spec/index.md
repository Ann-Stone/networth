# Spec Index — Balance Sheet

## Dashboard
- **File**: `dashboard.md`
- **Scope**: 首頁總覽、收支摘要、預算比較、財務目標、禮金紀錄、提醒通知

## Monthly Report
- **File**: `monthly-report.md`
- **Scope**: 月度記帳、現金流分析、支出比例、投資比例、預算對比、負債變動、餘額快照

## Year Report
- **File**: `year-report.md`
- **Scope**: 年度資產負債表、年度支出總覽、資產配置概覽

## Asset Management
- **File**: `asset-management.md`
- **Scope**: 股票資產、不動產、保險、其他資產的 CRUD 與估值管理

## Liability Management
- **File**: `liability.md`
- **Scope**: 貸款管理、還款紀錄、餘額追蹤

## Settings
- **File**: `settings.md`
- **Scope**: 帳戶管理、信用卡管理、支出分類（階層式）、預算設定、定期提醒、初始餘額設定

## Cross-Cutting
- **File**: `cross-cutting.md`
- **Scope**: 共用元件、下拉選單資料源、全域功能、資料匯入、API 架構調整等跨功能項目
