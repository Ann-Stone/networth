## Notes

- Reference `template.html` (repo root) for the exact hex values. Do not invent colors.
- Tailwind v4 in this project uses `@import "tailwindcss/utilities"` (not a config file `theme.extend.colors`). Custom utilities must be added via `@layer utilities` in the CSS file.
- `html.dark` class is toggled by the app (see `useAppStore` locale/dark toggle — add dark mode toggle if not present). Dark mode is opt-in; light mode is default.
- Do not change the `--el-border-radius-*` values.
- `.text-positive`, `.text-negative`, `.text-neutral` helper classes can remain unchanged (green/red/gray are universal).
