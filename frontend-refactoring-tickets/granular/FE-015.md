## Notes

- All components use `<script setup lang="ts">` and Composition API only.
- These are stateless prop-driven primitives — they emit nothing and hold no state.
- `MoneyDisplay` should handle negative numbers gracefully: show abs value with negative sign, not parentheses. Use the `positive` prop to control color, not the sign of `amount` (since expenses are stored as negative amounts in some contexts).
- Do not use `!important` to fight Element Plus styles.
