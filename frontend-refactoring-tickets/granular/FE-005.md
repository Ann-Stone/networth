## Notes

- `vesting_month` is a string in format `YYYYMM` (e.g. `"202601"`).
- `POST /monthly-report/stock-prices` accepts a JSON body matching `StockPriceHistory` (`stock_code`, `fetch_date`, OHLC fields) plus optional `trigger_yfinance: boolean`. There is no file upload at this endpoint; bulk CSV ingestion lives at `POST /utilities/import/stock-prices` and uses a separate `{ period }` body.
- Read `api/docs/api-reference.md` Monthly Report section for exact Journal, JournalExpenditureBudget, etc. shapes.
- `settleMonth` triggers the month-end balance settlement. It may take a moment; show loading state in the view.
