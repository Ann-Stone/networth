## Notes

- Charts are **stateless** — they receive all data via props and emit nothing. Views are responsible for fetching data and passing it down.
- Use `autoresize` prop on `<v-chart>` to handle container resize automatically.
- Do not use `echarts/dist/echarts` full bundle — always import specific modules for tree-shaking.
- If `vue-echarts` version 8 has API differences from v7, read its npm docs before implementing.
