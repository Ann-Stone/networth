## Notes

- Main code `code_id` is a user-supplied string PK (e.g. `"E01"`). It is used in PUT/DELETE URL, not the autoincrement id.
- `code_type` values: `Floating`, `Fixed`, `Invest`, `Income`, `Transfer` — render as select options.
- When a main code with `code_type` of `Fixed` or `Floating` is created, the backend auto-inserts a Budget row. The frontend does not need to handle this explicitly.
