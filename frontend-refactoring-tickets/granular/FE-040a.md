## Why this ticket exists

Phases 4 / 6 / 7 each ran a session against an empty dev DB and could
only verify type-check / build / color grep. Real CRUD round-trips,
chart renders, expand-row flows, and dashboard widgets were never
exercised in a browser.

Now that FE-040 has shipped fixtures, walk through each view, perform
the round-trips that were skipped, and capture the result. Fixtures are
deterministic, so anything that breaks here is a real bug — not a data
problem.
