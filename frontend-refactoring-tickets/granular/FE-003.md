## Notes

- All functions call `request.get/post/put/delete(url, payload)` — never call `axios` directly.
- The return type of each function is the unwrapped payload (request interceptor already strips the envelope), e.g. `Promise<Account[]>` not `Promise<ApiResponse<Account[]>>`.
- `PUT /settings/accounts/{id}` takes `AccountUpdate` (all fields optional). Pass only changed fields.
- `PUT /settings/budgets` takes `BudgetRead[]` array (bulk update). The endpoint URL is `/settings/budgets` with a PUT, not `/settings/budgets/{year}`.
- Do not add any functions beyond what the API spec provides.
