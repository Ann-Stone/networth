## Notes

- `vesting_month` format is `YYYYMM` (e.g. `"202601"`). `el-date-picker type="month"` returns a `Date`; convert with `dayjs(date).format('YYYYMM')`.
- Journal `action_main_type` values are `'Floating'`, `'Fixed'`, `'Income'` etc. (backend sends whatever is stored). Group logic for totals: income vs expense is based on `action_main_type`.
- CRUD buttons (add/edit/delete) are added in FE-022.
