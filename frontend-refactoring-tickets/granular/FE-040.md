| # | Task | Target file | Signature / key content | Acceptance command |
|---|------|-------------|-------------------------|--------------------|

## Notes

- MSW v2 API uses `http` (not `rest`) and `HttpResponse`. Import from `msw` not `msw/node`.
- The service worker file (`mockServiceWorker.js`) is generated by MSW CLI — do not hand-edit it. It must be in `public/` to be served at the root path.
- Mock data needs to be realistic enough to exercise all the UI code paths — include at least 2–3 rows per resource.
- The `worker` export from `handlers.ts` is created by `setupWorker(...handlers)` from msw.
